zhparser是什么</h1>
zhparser</a>是一个PostgreSQL</a>中文分词的插件，通过它，可以使PostgreSQL支持中文的全文检索（Full Text Search</a>）。

为什么需要zhparser</h1>
一般英语等语言分词比较简单，按照标点、空格切分语句即可获得有含义的词语，PostgreSQL自带的parser就是按照这个原理来分词的，比较简单。而中文就比较复杂，词语之间没有空格分割，长度也不固定，怎么分词有时还跟语句的语义有关，因此PG自带的parser不能用来做中文分词。使用zhparser这个插件，便可以使PG支持中文分词，继而可以使用PG做中文全文检索。

zhparser原理是什么</h1>
zhparser用C语言实现了PostgreSQL TEXT SEARCH PARSER</a>需要的接口，这些接口会调用SCWS</a>中文分词引擎进行分词。

zhparser使用手册</h1>

安装zhparser插件</h2>
1.安装SCWS

wget -q -O - http://www.xunsearch.com/scws/down/scws-1.2.1.tar.bz2 | tar xjf - cd scws-1.2.1 ;./configure ; make install </code>

2.下载zhparser源码

git clone https://github.com/amutu/zhparser.git </code>

3.编译和安装zhparser

SCWS_HOME=/usr/local make && make install </code>

注意：如果在*BSD系统上进行编译安装，请使用gmake代替make</span>

5.创建extension

psql dbname superuser -c 'CREATE EXTENSION zhparser' </code>
